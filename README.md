# Edge AI Karaoke System 🎤

## 📌 Description

This project is a real-time audio processing system that separates vocals and music using DSP and AI techniques, designed for edge devices.

## 🚀 Features

* Real-time processing
* Vocal separation
* Low latency
* Optimized for embedded systems

## 🛠 Technologies

* C / Python
* DSP (FFT, filtering)
* Embedded Systems

## ⚙️ How to Run

1. Load audio file
2. Run processing script
3. Output separated audio

## 📈 Future Work

* Improve model efficiency
* Deploy on embedded hardware
